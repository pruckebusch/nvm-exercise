Detailed Description

The general purpose NVM library (gp-nvm-lib) allows store and obtain attributes from non-volatile memory devices.

The gp-nvm-lib can be used to persistenly store attributes on a non-volatile memory device. For this purpose a getter and a setter function are provided. The gp-nvm-lib maintains a list of attributes in RAM, indexed by the attribute ID. Each entry contains the offset in the NVM and length of the attribute.
